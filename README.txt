# Hotel Sierra Azul — Página estática (Parte 1)
Cómo usar:
1) Descomprime el ZIP.
2) Abre `index.html` con doble clic en tu navegador.
3) Edita textos/precios/imagenes en `index.html` y estilos en `styles.css` si quieres personalizar.

Estructura:
- `index.html` — HTML5 semántico (sin Bootstrap).
- `styles.css` — CSS3 responsivo, grid, topbar sticky, cards.
- `main.js` — menú móvil y validación mínima del formulario.

Sugerencias:
- Reemplaza imágenes por fotos propias.
- Añade favicon y prueba con Lighthouse.
